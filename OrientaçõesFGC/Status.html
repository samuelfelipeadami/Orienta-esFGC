• Se o status for "Dados inconsistentes", será necessário pedir para um atendente nível 2
alterar o status no SPG para que o credor envie o documento novamente pelo app (Idwall). Responder:

"Prezado(a), o serviço do APP não faz leitura de RG funcional e passaporte,
somente RG nacional, CNH. Assim, aguarde até 48h para reprocessarmos sua solicitação para que tente com um documento válido.
Se não for possível, retorne neste canal que encaminharemos para a equipe que está tratando o pagamento fora do aplicativo."

• Se o status for Análise do Banco Pagador, responder:

"Prezado(a), verificamos no sistema que seu processo se encontra com o status de análise do retorno recebido do banco pagador,
que indica que houve a tentativa de transferência, mas o pagamento não foi concluído.
Nossa equipe está atuando. Se o problema for relacionado a dados bancários incorretos,
o sistema irá liberar o acesso para que complete os dados novamente para nova tentativa de transferência.
Se os dados estiverem corretos, o sistema tentará realizar a transferência dos recursos para a conta indicada novamente.

  Atenciosamente,
  Atendimento FGC."

Caso tenha alguma dúvida ou dificuldade responda esse e-mail com o print da mensagem que aparece para você no app do FGC.

 Atenciosamente,
  Atendimento FGC."
